► Definition & Purpose

Output styles are custom response formatting configurations for Claude Code CLI that control how Claude communicates and structures responses. Think of them as personality templates that modify Claude's behavior, tone, and presentation style.

► How They Work

System Prompt Modification: ┌─────────────────────────────────────┐ │ Base Claude Prompt + Output Style │ │ Instructions = Modified Behavior │ └─────────────────────────────────────┘

Each style appends specific instructions to Claude's system prompt, fundamentally changing how responses are structured and delivered.

► File Structure & Location

Global Styles: ~/.claude/output-styles/ Project Styles: {project}/.claude/output-styles/ Format: Markdown with YAML frontmatter Required Fields: description (in frontmatter) Precedence: Project > Global

► Style Capabilities

  • Response Format: Control structure (bullets, YAML, HTML, prose)
  • Communication Tone: Formal, casual, technical, educational
  • Information Density: Concise, detailed, comprehensive
  • Task Approach: Speed vs thoroughness balance
  • Code Style: Preferences for code formatting and comments
  • Workflow Behaviors: Tool usage patterns and step sequences
