► Quick Definition

Output styles are response templates that control how Claude formats and delivers answers in Claude Code CLI. They're like switching between different AI personalities or communication modes.

► Core Concept

► Technical Mechanism

Each style is a markdown file containing instructions that get appended to Claude's system prompt. This modifies behavior at the AI model level.

File Structure: ├── ~/.claude/output-styles/ (Global styles) └── project/.claude/output-styles/ (Project styles) Format: Markdown with YAML frontmatter Required: description field in frontmatter
